# prompt-examples-web
A website with AI prompt examples. Prompt repository

The Oflameron text programming language is a language developed and enhanced using AI prompts. It is designed for encryption, text obfuscation, and steganography. Using example prompts, you can convert encrypted text into program source code in any language, scripts, nonsense text, and mathematical formulas. Using AI prompts, you create a custom version of the Oflameron language each time, making cryptanalysis more difficult. AI prompts allow you to easily and quickly modernize the language and add features—one of the basic versions is provided here. You can develop your own clone of the language using the MIT License.
